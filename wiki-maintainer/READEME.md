# Wiki Maintainer

This is the wiki maintainer agent. It is responsible for all edits to the wiki.

You can see technical details of how it works and is structured in [overview.md](overview.md).

## Examples

You can open Claude Code in this directory and say things like:

"I want to add a new question to the wiki: when would I want to use a subagent rather than putting instructions in CLAUDE.md?"

"Can you add a new tool page to the wiki for the Pi agent harness?"
